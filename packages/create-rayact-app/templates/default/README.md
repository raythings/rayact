# __PROJECT_NAME__

Rayact app — React UI on QuickJS + raylib.

## Get started

```bash
npm install
npm run dev
```

In another terminal:

```bash
npm run start:dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start Rayact dev server (bundler + HMR) |
| `npm run start` | Run desktop host with built `dist/bundle.js` |
| `npm run start:dev` | Run desktop host connected to dev server |
| `npm run build` | Production Vite build → `dist/bundle.js` |
| `npm run android` | Configure adb reverse for Android dev |
