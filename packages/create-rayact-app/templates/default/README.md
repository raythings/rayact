# __PROJECT_NAME__

Rayact app — React UI on QuickJS + raylib.

## Get started

```bash
npm install
npm run dev
```

In another terminal:

```bash
npm run start:dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start Rayact dev server (bundler + HMR) |
| `npm run android` | Install + launch the prebuilt Android dev app |
| `npm run ios` | Install + launch the prebuilt iOS simulator dev app |
| `npm run prebuild` | Materialize native Android/iOS shells linked to prebuilts |
| `npm run android:dev-client` | Build + install a custom Android debug client |
| `npm run ios:dev-client` | Build + install a custom iOS simulator debug client |
| `npm run build:desktop` | Build a self-contained desktop release |
| `npm run build:android` | Build Android release |
| `npm run build:ios` | Build iOS release |
| `npm run build:web` | Build web release |
| `npm run start:dev` | Run desktop host connected to dev server |
