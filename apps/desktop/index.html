🚀 Key Features

⚡️

QuickJS Integration

Embedded JavaScript runtime for native execution without node.js dependencies.

⚛️

React 19

Real react-reconciler 0.31 host config — hooks, suspense, and react-navigation.

🎨

raym3 + raylib

Material components with real Yoga flexbox layout, raylib-backends acceleration on Android.

💾

Persistent Storage

Built-in sync KV + AsyncStorage with write-behind flush. Opt-in MMKV and Keychain/Keystore secure store as autolinked plugins.

🔌

Native Module Bus

Cross-context native-function registry. dlopen plugins autolink at runtime — no compile flags, no engine rebuild.

🧵

Workers + WASM

JS and WASM workers as a near-native compute lane — and they reach the same native modules via sys_invoke.

📱

Cross-Platform

One codebase shipping on Desktop (macOS/Windows/Linux) and Android; Web via raygpu planned.

💻 Quick Start

Build and run the rayact desktop application in minutes

# Build dependencies cd ../quickjs && meson setup build && meson compile -C build cd ../raylib && cmake -B build -DPLATFORM=Desktop && cmake --build build cd .. # Build and run cmake -B build -S . -DENABLE_DESKTOP=ON cmake --build build ./build/bin/rayact_desktop
